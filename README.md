## Multi-Agent-NL-to-SQL-Visualization-System
# 📌 Overview:
This project allows users to interact with a database using simple English instead of writing SQL queries. It uses a multi-agent system to understand user input and provide accurate results and visualizations.

# 🧠 Features:
- 🔐 Authentication for secure access
- 🧭 Router to classify user intent
- 💬 Chat & General AI response
- 🕘 History node for past queries
- 🧾 Natural Language to SQL conversion
- 📊 Python-based data visualization
- 🖥 Streamlit UI for user interaction

## ⚙️ Tech Stack
- Python
- LangGraph
- Streamlit
- DuckDB
- Pandas
- Matplotlib

## 🔄 Workflow
- User enters query
- System authenticates user
- Router decides the task
- Query goes to correct node (chat, SQL, history, etc.)
Result is shown as table or chart

🎯 Use Case
Helps non-technical users to easily query and analyze data without SQL knowledge.
